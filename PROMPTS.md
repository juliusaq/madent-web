# Prompts Reutilizables — MADENT

Este documento contiene una biblioteca de prompts optimizados para el uso de modelos de lenguaje e inteligencias artificiales que interactúen con el proyecto.

---

## 📌 Prompt: Diseño y Construcción de Interfaces

```text
Actúa como un desarrollador frontend experto. Construye interfaces web premium para el proyecto MADENT.

### Estilo Visual
Inspírate en la estética de Apple, Stripe, Notion y Linear.
- Mucho espacio en blanco (respiro visual, márgenes y rellenos amplios).
- Bordes suaves (esquinas con rounded-2xl o rounded-3xl).
- Sombras sutiles (sombras finas shadow-sm o shadow-md con baja opacidad).
- Enfoque mobile-first.
- Animaciones discretas (duraciones cortas, efectos hover refinados).
- Cumplimiento de accesibilidad web (contraste de color, etiquetas semánticas y legibilidad).
- Alto rendimiento (código limpio, sin dependencias externas innecesarias).

### Paleta de Colores Oficial
- Primary: #0F172A (Azul pizarra profundo)
- Accent: #14B8A6 (Turquesa / Teal)
- Background: #FAFAFA (Gris ultra claro)
- Text: #64748B (Slate gris para textos secundarios)

### Objetivo
Generar interfaces orientadas a la conversión, motivando a los odontólogos independientes a solicitar una demostración del producto.
```
